学习MarkDown语法


1.标题书写形式

两种标题的语法，分别是Setext和atx形式。

  1. Setext是通过“=”和“-”分别对应最高阶和第二阶。
  2. atx是通过行首“#”的个数来区分标题阶数。ps.记得空格

2.列表书写形式

列表可以使用星号、加号、减号来作为列表标记。

* candy

* Gum

* Booze

有序的列表如下：

1. Red

2. Green

3. Blue

3.链接的表示方式

Markdown支持两种形式的链接语法：行内 和 参考 两种形式。

行内：

this is an[example link](http://example.com/).

要注意的是分清圆括号和方括号。

参考:

I get 10 times more traffic from [Google][1] than from
[Yahoo][2] or [MSN][3].

[1]: http://google.com/ “Google”

[2]: http://search.yahoo.com/ “Yahoo Search”

[3]: http://search.msn.com/ “MSN Search”

同样要注意上面用的全是方括号

4.图片的表示方式

图片的语法和链接的基本一致，同样分为行内和参考两种形式。

行内：

![alt text](/path/to/img.jpg “title”)

其中alt text是替代文字，当照片无法显示的时候会显示这行替代文字。
圆括号内前面的为src，后面为标题。

参考：

![alt text][id]

[id]:/path/to/img.jpg “title”

前后都是方括号，图片链接的话通过图床可以简单获取。

title可写可以不写

5.代码的表示方式

使用反引号`来标记代码区段，会自动把html换成html实体。在贴html代码的时候很方便。

I strongly recommend against using any `<blink>` tags.

更方便的方式就是在每行的行首缩进四个空格或者一个Tab，便可以获得一个格式化好的代码区块。

6.强调的表示方式

使用星号、底线作为标记强调字词的符号。

斜体的表示方式为：

*sentence*

_sentence_

粗体的表示方式为：

**sentence**

__sentence__
