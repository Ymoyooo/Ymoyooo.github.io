学习数据库

  1. 初始化数据库mysqld --initialize --console
  2. 启动MySQLnet start mysql
  3. 登录本机数据库mysql -u root -p
  4. 密码：tNohu1taOy)I
  5. 启动mysqld --console
  6. shutdown mysqladmin -uroot shutdown
  7. 查询表内容desc xxx(表名字)
  8. DATETIME 混合日期和时间值
  9. DATE 日期值
  10. 创建数据库creat database name;
  11. 删除数据库drop database name;
  12. 对某库进行操作use name(库名);
  13. 创建一个表CREATE TABLE name
  14. 修改 表名ALTER　table name(现有表名) rename name2(修改后的表名)
  15. 在表内增加字段ALTER TABLE name(表名) add column NAME2(size)(字段名称+长度)
  16. 删除字段ALTER TABLE NAME drop column NAME2
  17. 修改字段名ALTER TABLE NAME change column name1(改前) name2(改后) 字符类型
  18. 修改字段名alter table tname change name1 name2 字段类型+长度 after name3
  19. 修改行内信息update tname set parameter='..',parameter2='..' where parameter='...' p,p2是要修改的项，p是我们要查找的那一行数据key
  20. 添加行信息 insert into tname(p1,p2,p3,...,pn) value('v1','v2',...,'vn');
  21. 删除记录 DELETE FROME 表名 WHERE 条件 delete frome emp3 where id='1';
  22. 查找数据 SELECT FROM 表名`select from emp where id=’1’`
  23. 按条件查找数据 select a,b frome emp where salary>1000
